[Linux]遇到的问题和一些简单的常用的笔记
一、常用的Linux命令
	显示当前目录下的文件或目录：
		ls -l：显示长列表
		   -h：使用人性化的方式显示文件的大小[显示文件的M字节或者K字节，而不只是显示字节数]
		   -a：显示所有的文件[包括隐藏文件或文件夹]
		   -i：显示文件的节点号
		   -d：显示目录本身的信息
	显示安装RPM文件的过程、日志和时间等详细信息：
		rpm -ivh rpm包名
	查询所有安装的RPM软件中是否安装过mysql：
		rpm -qa | grep -i mysql
	卸载RPM包：
		rpm -e rpm包名
	用来查看当前Linux操作系统是否存在该用户：
		id + 用户名[例如简单的root]
	启动或者关闭Linux服务：
		service 服务名 start
		service 服务名 stop
	为Mysql的root用户设置密码：
		/usr/bin/mysqladmin -u root paaword 12345
	查看Mysql的安装版本：
		mysqladmin --version
	将某个软件设置开机启动：
		chkconfig mysql on[使用时将mysql替换成为其他的软件即可]		
	查看所有开机自启的软件：
		chkconfig --list
	查看某个进程是否是在运行的状态：
		ps -ef | grep 进程的名字
	查看当前Linux服务器中存储介质的状态：
		df -h	
	创建目录：
		mkdir + 目录名[上级目录必须存在]
		mkdir -p + 目录名[迭代的创建目录]
	切换目录：
		cd + 切换目录
	查看当前所在的目录：
		pwd
	删除目录或文件：
		rmdir + 目录[删除目录]
		rm -r + 目录或文件名[删除目录或者文件名，不加-r只能删除目录]
		rm -f + 目录或文件名[强制删除目录或文件名]
		所以一般使用的都是 rm -rf + 指定的文件或文件夹
	显示当前系统的时间：
		date
	复制文件或目录：
		cp -r + 目录或文件[不加-r只能复制文件]
	剪切或者改名：
		mv 源文件 目标目录
		如果源文件和目标文件在同一个目录下就是改名字[即如同Windows下的重命名]，否则就是剪切

四、Vim编辑命令
	Vim同时编辑多个文件：
		vim -o file1 file2 [横向显示多个要打开的文件]		
		vim -O file1 file2 [纵向显示多个要打开的文件]
	在光标之前插入字符：
		i
	在光标之后插入字符：
		a
	跳转到打开文件的最后一行：
		G
	跳转到打开文件的头部：
		gg
	从当前光标之后进行指定字符的查找：
		/要查找的字符
	从当前光标之前进行指定字符的查找：
		?要查找的字符	
	当进入查找的模式的时候，可以进行上一个目标和下一个目标的切换[此切换是相对于查找的方向来的]：
		n[查找下一个]
		N[查找上一个]
	为当前打开的文件设置行号：
		:set nu
	为当前打开的文件取消行号：
		:set nonu
	保存文件:
		:w
	退出并且不保存文件：
		:q!
	退出并保存文件：
		:wq
	移动到文件的最后一行：
		:$
	移动到文件的指定行：
		:n
	将光标进行移动：
		h[向左] j[向下] k[向上] l[向右]
	将光标移到当前行的末尾：
		shift+$
	在已经打开文件的情况下打开其他的文件：
		:vs 文件路径+文件名[横向展示文件]
		:sq 文件路径+文件名[纵向展示文件]
	在多个文件打开的情况下进行不同窗口的切换：
		ctrl + ww
	多个文件打开的情况下：
		:qa[关闭所有文件]
		:only[只剩下当前光标在的文件，关闭其他文件]
	常用的git命令：
		git status
		git config
		git add .
